# Performance Regression Template

> **[必填]** 标记为必须填写，其余按需填写。
> 术语定义见 [`_glossary.md`](./_glossary.md)。
> **必填时机说明**：`[创建]` = 创建时必须填写；`[结束]` = 关闭前必须填写；未标注 = 按需填写
> 状态流转见 `_glossary.md` → 对应类型。
>
> **用途**：用于跟踪性能下降问题，从指标异常到根因定位到修复验证的全流程。
> **注意**：如果是功能性的性能优化（非回归），请改用 feature 模板。
>
> **升级规则**：见 `_glossary.md` → Quick Fix 升级规则。性能问题涉及多文件时，关闭 quickfix，改用本模板。
