# Tech Debt Template

> **[必填]** 标记为必须填写，其余按需填写。
> 术语定义见 [`_glossary.md`](./_glossary.md)。
> **必填时机说明**：`[创建]` = 创建时必须填写；`[结束]` = 关闭前必须填写；未标注 = 按需填写
> 状态流转见 `_glossary.md` → 对应类型。
>
> **用途**：用于记录和跟踪已知的技术债务，避免"暂时先这样"变成永久方案。
> **注意**：如果债务需要立即偿还（阻塞开发），请改用 refactor 模板。
>
> **升级规则**：见 `_glossary.md` → Quick Fix 升级规则。债务偿还涉及架构调整时，关闭 quickfix，改用 refactor 模板。
